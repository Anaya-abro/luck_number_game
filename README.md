# 🍀 Lucky Number Game

A simple Python game where you guess a number between 1 and 5 and try to match the computer's number.

## 🎮 Features

- Random number generation
- User input handling
- Win/Lose result
- Replay option
- Beginner-friendly code

## 🕹️ How to Play

1. Run the Python file
2. Enter a number between 1 and 5
3. If your guess matches the computer's number, you win 🎉
4. Otherwise, you lose 😟
5. Choose to play again or exit


python lucky_number_game.py
